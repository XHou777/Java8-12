# Java 8 特性回顾

该项目旨在探索和回顾自 2020 年 1 月 6 日起推出的 Java 8 的新特性。通过这个项目，我将深入研究 Java 8 带来的各种新特性，包括但不限于 lambda 表达式、流 API 和新的日期和时间 API。
